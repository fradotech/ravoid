# Lightweight Alternatives

- Easy onboarding
- Basic automation

- Pipeline-focused

# Decision Criteria

- Sales complexity
- Automation need
- Reporting requirement
- Budget

# Growth Advice

Early stage:
Simple pipeline tracking > automation.

Scale stage:
Move to structured CRM.

Complexity should follow revenue.
